# TUKIO PRO — Site Web Officiel

Plateforme digitale premium pour l'organisation d'e-votings payants, crowdfunding, billetterie corporate et événementielle privée.

## Structure du projet

```
tukio-pro/
├── home/                     → Page d'accueil
├── e-voting/                 → Module e-voting
├── crowdfunding/             → Module crowdfunding
├── corporate-ticketing/      → Billetterie corporate
├── private-ticketing/        → Billetterie privée
└── images/                   → Assets visuels
```

## Stack technique

- HTML5 sémantique
- CSS3 (animations, responsive mobile-first)
- JavaScript Vanilla (sans dépendances)

## Typographies

- **Posterama 1913 SemiBold** — Titres (via Adobe Fonts / CDN)
- **Montserrat** — Corps de texte (via Google Fonts)

## Palette

| Rôle        | Valeur    |

## Démarrage

Ouvrir `home/index.html` dans un navigateur moderne.
